Description:
This program simulates a SuperMarket's ordering and dispatch system. The program takes in orders and
information via an input file and dispatches them using a variety of data structures. The dispatch
information is stored as shipping labels and printed to an output file.

How To Compile:

-Run the 'make' command
-Run ./p1 'inputfile.txt', replacing 'inputfile.txt' with the name of whatever input text file
you'd like to run the program on
-The program's output will be written to output.txt in the working directory
